# 企业人事管理系统 python487

## 项目概述

企业人事管理系统是一款用于帮助企业高效管理员工信息的软件。该系统基于Python3、pyqt5和MySQL技术栈开发，提供员工基本信息、绩效信息以及工资信息的维护功能。

## 技术栈

- **后端**: Python3
- **前端**: PyQt5
- **数据库**: MySQL

## 功能模块

### 登录页面

- 用户身份验证

### 员工信息管理

- **增删改查**
  - 新增员工信息
  - 删除员工信息
  - 修改员工信息
  - 查询员工信息

### 绩效管理

- **增加及修改**
  - 新增绩效记录
  - 修改绩效记录

### 工资管理

- **增加及修改**
  - 新增工资信息
  - 修改工资信息

## 系统角色

- 系统管理员：负责系统的维护及用户权限分配
- 普通用户：执行日常的人事管理操作

## 运行环境

- 操作系统：兼容Windows、Linux和macOS
- Python版本：Python3及以上
- 数据库：MySQL（版本不限）

## 部署步骤

1. 安装Python3环境
2. 安装PyQt5库
3. 安装MySQL数据库，并创建相应数据库和表结构
4. 配置系统环境变量（如有需要）
5. 运行系统
